# IPCC color palette

automatically use the suggested IPCC colors for precipitation and
temperature.

## Usage

``` r
IPCC_palette(type, divergent)
```

## Arguments

- type:

  character, one of tmp or pr.

- divergent:

  logical. If TRUE, divergent palette are used. Useful in combination
  with legend.range to assign central colors in the palette to zero
  values
